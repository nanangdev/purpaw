# Purpaw - Cat Health

Purpaw is a comprehensive web-based app for pet cat care.

## Key features include:
- Cat health tracker.
- Find veterinarians & services.
- Grooming.
- Lost, Found, or Adobted cats.
- Games for you and your cat.
- Offline-first.
- And much more.
> Feature can scale and be expanded in the future.
